## 📄 Descrição dos Dados

Este repositório contém dados extraídos a partir do número do processo, permitindo a consulta das sentenças correspondentes.

Para cada registro, foram extraídas:
- **Sentença base**
- **Sentença definitiva**

As penas foram estruturadas de forma detalhada, sendo separadas em colunas distintas:
- anos  
- meses  
- dias  

Por exemplo, uma pena de **1 ano, 3 meses e 2 dias** é representada como:
1,3,2

---

## 🔎 Consulta por Processo

Cada processo pode ser localizado diretamente por meio do seu número.

Exemplo:
1500073-10.2021.8.26.0341


O arquivo correspondente pode ser encontrado como:
1500073-10.2021.8.26.0341.txt

## 📊 Origem dos Dados

Os arquivos CSV contêm dados extraídos do Tribunal de Justiça de São Paulo (TJSP), referentes ao ano de 2022.

Esses dados foram utilizados no artigo:  
**"Explaining the Judges’ Decisions Criteria" (KDIR 2025)**
